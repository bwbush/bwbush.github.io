Genetic Algorithms

This applet evolves a population of genotypes according to a customizable genetic algorithm. Example genotypes are provided for several problems:

  • banda.genalg.UnivariateFunctionExample - maximizing an example function of a single variable
  • banda.genalg.MultivariateFunctionExample - maximizing an example function of two variables
  • banda.genalg.DynamicControlExample - finding the optimal solution for a control system
  • banda.genalg.KnapsackExample - trying to fit the most valuable items from a set into a knapsack of limited capacity
  • banda.genalg.TravelingSalesmanExample - finding the shortest tour of a set of cities
  • banda.genalg.prog.UnivariateFunctionExample - approximating a function of a single variable

To switch problems, push the setup button below and enter the name of the new genotype class in the Genotype class field of the setup dialog box.

If your browser recognized the applet tag, you would see an applet here.

You can also use the Java Applet Viewer to run this applet: enter appletviewer http://www.bwbush.io/applets/banda/genalg/algorithm.html at the command line.
