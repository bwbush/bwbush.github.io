Eclipse Visualizer

The applet below shows a 3D visualization of solar and lunar eclipse data between the years 1975 and 2025. Use the buttons to navigate the scene and the mouse to query the objects. Each box represents a data record: the position, size, orientation, and color of the box represent attributes of the data.

(This applet requires Java 3D 1.2 or higher.)
